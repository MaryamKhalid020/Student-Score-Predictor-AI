# Student Score Predictor 🎓

A complete end-to-end Machine Learning project that predicts student exam marks based on study hours. This project features a **Linear Regression** model and a **Tkinter-based Graphical User Interface (GUI)** for real-time interaction.

## 🚀 Features
- **Predictive Modeling:** Uses Simple Linear Regression to model the relationship between study time and academic performance.
- **Interactive GUI:** Built with Tkinter to allow users to input hours and receive instant mark predictions and grade estimates.
- **Data Visualization:** Includes Matplotlib plots for Regression Fit and Residual analysis to verify model accuracy.
- **Logic Constraints:** Ensures predictions stay within a realistic academic range of 0–100%.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **ML Library:** Scikit-learn
- **Data Handling:** Pandas & NumPy
- **Visualization:** Matplotlib
- **GUI Framework:** Tkinter

## 📊 Model Evaluation
The model calculates key metrics to ensure high accuracy:
- **R² Score:** Measures how well the regression line fits the data.
- **Mean Absolute Error (MAE):** Quantifies the average prediction error.
- **Visuals:** Generates a regression plot and a residual plot to detect patterns in errors.
